# Kubernetes series

Link: https://youtube.com/playlist?list=PLRz-2ltlXLUJTE2tNvIWAzRAG-aqLpzZx&si=Zcq4mMLjLL9Mu637

## Modules

1. install.md (Cài đặt K8s Cluster chuẩn Production)
2. basic-command.md (Khái niệm và các lệnh kubectl cơ bản)
3. learn-yaml.md (Quản lý workload và tạo tài nguyên từ file YAML)
4. service-types.md (Triển khai ứng dụng và kết nối dịch vụ cơ bản)
5. secret-configmap-env.md (Thiết lập cấu hình và quản lý biến môi trường)
6. storage-types.md (Quản lý các loại lưu trữ và Persistent Volume)
7. publish-service-app.md (Công khai ứng dụng và dịch vụ ra Internet)
8. manage-context.md (Quản lý Context và môi trường làm việc hiệu quả)
9. advanced-scheduler.md (Kỹ thuật lập lịch và điều phối Pod nâng cao)
10. argocd.md (Triển khai ứng dụng tự động theo chuẩn GitOps)
11. rancher.md (Quản trị và vận hành Cluster tập trung với Rancher)
