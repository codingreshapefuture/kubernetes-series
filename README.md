# Kubernetes series

Link: https://youtube.com/playlist?list=PLRz-2ltlXLUJTE2tNvIWAzRAG-aqLpzZx&si=Zcq4mMLjLL9Mu637

## Modules

1. install.md (Cài đặt K8s Cluster chuẩn Production)
2. basic-command.md (Khái niệm và các lệnh kubectl cơ bản)
3. learn-yaml.md (Quản lý workload và tạo tài nguyên từ file YAML)
4. publish-service-app.md (Triển khai ứng dụng và kết nối dịch vụ cơ bản)
5. networking-and-services.md (Phân dải DNS và giao tiếp dịch vụ nội bộ)
6. storage-types.md (Quản lý các loại lưu trữ và Persistent Volume)
7. secret-configmap-env.md (Thiết lập cấu hình và quản lý biến môi trường)
8. ingress.md (Quản lý Traffic và Routing với Ingress)
9. advanced-scheduler-1.md (Kỹ thuật lập lịch và điều phối Pod nâng cao P1)
10. advanced-scheduler-2.md (Kỹ thuật lập lịch và điều phối Pod nâng cao P2)
11. advanced-scheduler-3.md (Kỹ thuật lập lịch và điều phối Pod nâng cao P3)
12. cloud-k8s-engine.md (Giới thiệu các nền tảng K8s trên Cloud)
13. rancher.md (Quản trị và vận hành Cluster tập trung với Rancher)
14. helm.md (Quản lý gói ứng dụng với Helm Chart)
15. argocd.md (Triển khai ứng dụng tự động theo chuẩn GitOps)
